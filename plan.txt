# Harness Development Plan

## In Progress

### TODO: Fix MiniMax tool calling format in brain.py
The MiniMax API requires tools to be formatted as:
```json
{"type": "function", "function": {...}}
```

But brain.py currently passes tools as-is to all providers. We need to:
1. Detect when provider_type is "minimax"
2. Automatically wrap each tool in `{"type": "function", "function": ...}` before sending

This is required for `cloud-pro` (MiniMax-M2.7) to work with tool calling.
